# Project Instructions: GMAO Pro

## Localization & Language
- **Primary Language:** The entire application MUST be in **English**. This includes all UI text, labels, messages, and documentation.
- **Date Format:** Use English/US date formats (e.g., `MM/DD/YYYY` or `YYYY-MM-DD` depending on context, but consistently English-styled).
- **Time Format:** Use 12-hour or 24-hour format as appropriate for English locales.

## Development Standards
- All new code and modifications must adhere to the English-only rule.
- Ensure `includes/lang.php` and translation functions support English as the default and only language if necessary.
